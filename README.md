# wenagoo

Plateforme de voyage et de réservation (Next.js 16 + React 19 + TypeScript).

> Base UI dérivée du template Travila (AliThemes). Sources originales (doc + Figma) archivées dans `../archive/`.

## Démarrage

```bash
npm install
npm run dev
```

## Scripts

- `dev` — serveur de développement
- `build` — build production
- `start` — démarre le build
- `lint` — ESLint
- `sass` — watcher SCSS → CSS
